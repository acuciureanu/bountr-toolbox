### Prerequisites

[golang](https://golang.org/doc/install)

After installing Go I add the following lines to:

`~/.bash_profile`
```sh
export GOPATH=$HOME/go
export PATH=$PATH:/usr/local/go/bin:$GOPATH/bin
```

`~/.bashrc`
```sh
 if [ -f ~/.bash_profile ]; then
    source ~/.bash_profile
 fi
```

### Run the script

```sh
$ bash bountr-toolbox.sh
```
